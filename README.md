# Amorphous-LLC Amorphous

## How do I install these formulae?

`brew install amorphous-llc/amorphous/<formula>`

Or `brew tap amorphous-llc/amorphous` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "amorphous-llc/amorphous"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
